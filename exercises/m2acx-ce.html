若數字可以被 2 整除，它一定可以用「 2 個一排」的方式排列整齊。

舉例來說， 8 可以被 2 整除，而且 8 也可以排列成：

init( { range : [ [ -6, 6 ], [ -2, 2 ] ], scale : 40 } ); circle( [ -5, -1 ], 0.5 ); circle( [ -3.5, -1 ], 0.5 ); circle( [ -2, -1 ], 0.5 ); circle( [ -0.5, -1 ], 0.5 ); circle( [ -5, 1 ], 0.5 ); circle( [ -3.5, 1 ], 0.5 ); circle( [ -2, 1 ], 0.5 ); circle( [ -0.5, 1 ], 0.5 );

所以「若數字可以被 2 整除，那這個數字一定是偶數」。

所以「若數字被 2 除餘 1 ，那這個數字一定是奇數」。
