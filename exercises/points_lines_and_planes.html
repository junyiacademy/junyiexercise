任何平面都可以用三個不共線的點表示

不共線的點，就是不在同一條直線上的點。

在平面 \mathcal{R} 找出不在同一條直線的任何三點，並將它們列出來 (順序沒有關係) 。

例如，我們可以把平面 \mathcal{R} 寫成 平面 A + B + E 、平面 A + C + E、或平面 B + E + C 。

SOLUTION
  • INCORRECT_1
  • INCORRECT_2
  • INCORRECT_3
  • \overleftrightarrow{A + B + C}

任何一條直線都可以用直線上的任意兩點來定義。順序沒有關係。

點的上方必須加上 \leftrightarrow 的符號，這樣才能清楚表示我們在定義的是一條直線，而不是射線或線段。

所以，另外一個表示直線 \ell 的方法是 SOLUTION 。
