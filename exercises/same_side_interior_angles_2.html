粉紅色的角在藍色的角旁邊，而且兩個角合起來組成一條直線，所以我們知道：

\color{BLUE}{Ax + B} + \color{PINK}{y} = 180

兩個粉紅色的角是相等的，因為它們是 \color{GREEN}{\text{對頂角}} 。

graph.pl.drawAdjacentAngles( KNOWN_INDEX, "y^\\circ", "#FF00A5" );

其中一個粉紅色的角是橘色角的 \color{GREEN}{同位角}，而另一個粉紅色的角則是橘色角的 \color{GREEN}{\text{內錯角}}。因此，橘色角的角度跟粉紅色角是相等的。

\color{PINK}{y} = \color{ORANGE}{Cx + D}

把 \color{PINK}{y} 用 \color{ORANGE}{Cx + D} 代入到我們的第一個等式中。

\color{BLUE}{Ax + B} + \color{ORANGE}{Cx + D} = 180

合併同類項。

A + Cx + B + D = 180

將兩邊同時B + D > 0 ? "減去" : "加上" \color{PINK}{abs(B + D)} 。

(A + Cx + B + D) \color{PINK}{+ -(B + D)} = 180 \color{PINK}{+ -(B + D)}

A + Cx = 180 - B - D

將兩邊同時除以 \color{PINK}{A + C} 。

\dfrac{A + Cx}{\color{PINK}{A + C}} = \dfrac{180 - B - D}{\color{PINK}{A + C}}

化簡。

x = (180 - B - D) / (A + C)

另外，你會注意到藍色的角和橘色的角互為補角。
