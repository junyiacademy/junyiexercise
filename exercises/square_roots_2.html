兩個整數, 例如 6

想想有哪些完美平方數靠近 Q. [什麼是完美平方根?]

將一個整數平方後, 就可以得到一個完美平方數

而前13個完美平方數是:

\qquad 1,4,9,16,25,36,49,64,81,100,121,144,169

N * N 是小於Q中最靠近的完美平方數

(N + 1) * (N + 1) 是大於Q中最靠近的完美平方數

所以, 我們可以得到 N * N < Q < (N + 1) * (N + 1).

而 \sqrt{N * N} < \sqrt{Q} < \sqrt{(N + 1)*(N + 1)}.

所以 \sqrt{Q} 就介於 N 和 N + 1之間
