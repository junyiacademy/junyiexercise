有一等差數列，其第 n 項公式為 a_n = A+ D\times n，試求其公差為何？

D

公差即後項減前項之差，試著由 a_n = A+ D\times n 代入 n 為 1, 2, 3 來感受 a_n 的變化。

a_1= A + D
a_2= A + 2*D
a_3= A + 3*D

可觀察到，每多一項其數量改變 D。
要特別留意， n 的係數反映了公差。
此題答案為 D。
