randRangeUnique(100, 999,3) A+B+C

花園裡有 A 朵紅玫瑰、B 朵白玫瑰跟 C 朵黃玫瑰。請問花園裡共有幾朵玫瑰？

ANS

紅玫瑰和白玫瑰共有 A + B =A+B 朵。

紅玫瑰和白玫瑰的數量 + 黃玫瑰的數量 = 玫瑰的數量

所以花園裡共有 A+B + C = ANS 朵玫瑰。

randRangeUnique(100, 650,3) A+B+C

小琪買了一條 A 元的項鍊、一雙 B 元的鞋子跟一個 C 元的皮包，請問小琪總共花了多少錢？

ANS

項鍊和鞋子總共 A + B =A+B 元。

項鍊和鞋子的錢 + 皮包的錢 = 小琪花的錢

因此小琪總共花了 A+B + C = ANS 元。

randRangeUnique(100, 250,3) A+B+C

便當店中午賣出了 A 個排骨便當、B 個雞腿便當跟 C 個招牌便當，請問便當店共賣出了幾個便當？

ANS

排骨便當和雞腿便當共賣出了 A + B =A+B 個。

排骨便當和雞腿便當的個數 + 招牌便當的個數 = 全部便當的個數

因此便當店共賣出了 A+B + C = ANS 個便當。

randRangeUnique(200, 250,3) A+B+C

均一國中一年級有 A 位學生、二年級有 B 位學生、三年級有 C 位學生，請問均一國中共有幾位學生？

ANS

一年級和二年級共有 A + B =A+B 位學生。

一年級和二年級的學生數 + 三年級的學生數 = 全校的學生數

因此均一國中共有 A+B + C = ANS 位學生。
