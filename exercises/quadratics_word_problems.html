有一位K想用 J*100 公尺的籬笆圍成一個矩形的菜園，問如何圍出最大面積的菜園？並求出此面積。

(J*100/4)*(J*100/4)平方公尺

如下圖，設農夫所圍成矩形菜園的一邊長為 x 公尺。由題意知另一邊長為 (J*100/2-x) 公尺。

init({ range: [ [0, 10], [0, 5] ], scale: 40 }); path([[1,1],[6,1],[6,4],[1,4],[1,1]]); label([3.5,0.5],J*100/2+"-x","center"); label([0.5,2.5],"x","center");

因此菜園的面積為

x(J*100/2-x)=J*100/2x-x^2

=-(x^2-J*100/2x)

=-(x^2-2\cdotJ*100/4x+J*100/4^2-J*100/4^2)

=-(x-J*100/4)^2+(J*100/4)*(J*100/4)

因此當 x=J*100/4 時，面積有最大值 (J*100/4)*(J*100/4) 。此時菜園的一邊長為 J*100/4 公尺，另一邊長為 J*100/2-J*100/4=J*100/4 （公尺），也就是說當此菜園是正方形時，其面積最大，且面積為 (J*100/4)*(J*100/4) 平方公尺。

所以，本題答案為 (J*100/4)*(J*100/4) 平方公尺。

Name參加創意Team，製作了一個小火箭，完成後拿到操場試射。 假設小火箭沿二次函數 y=-\dfrac{1}{10*K}x^2+\dfrac{M}{5}x+N 的軌跡飛行，如下圖：

init({ range: [ [0, 10], [0, 5] ], scale: 40 }); line([1,0],[1,5]); line([1,5],[0.8,4.8]); line([1,5],[1.2,4.8]); line([0,1],[9,1]); line([8.8,1.2],[9,1]); line([8.8,0.8],[9,1]); arc( [1.7, -1], 4, 30, 100, { stroke: "black" } ); label([1.7,2.5],"(0,"+N+")","center"); label([0.5,4.5],"y","center"); label([8.5,0.5],"x","center");

其中 x 公尺為火箭飛行的水平距離，y 公尺則為火箭距地面高度，發射點在 (0,N) 的位置。試回答下列問題：

(1)火箭從發射到落地時，飛行的水平距離為多少公尺？

(2)在飛行過程中，火箭離地面的高度最高為多少公尺？

落地水平距離： Res公尺

最大高度： Result公尺

整數，例 6

小數，例 0.6

(1)我們必須知道火箭落地的位置。當火箭落地時，高度為 0 ，即 y=0，所以可得一元二次方程式：

-\dfrac{1}{10*K}x^2+\dfrac{M}{5}x+N=0

化簡得 x^2-2*K*Mx-10*K*N=0

由因式分解得 (x-Res)(x-Res2)=0

解一元二次方程得 x=Res，或 x=Res2（不合題意）

因此，火箭落地的地點到發射點的水平距離為 Res-0=Res 公尺。

(2)因為 y 公尺表示火箭的飛行高度，火箭離地面最高的高度即為函數

y=-\dfrac{1}{10*K}x^2+\dfrac{M}{5}x+N 的最大值，所以，可將此函數配方而得到：

=-\dfrac{1}{10*K}(x^2-2*K*Mx)+N

=-\dfrac{1}{10*K}(x^2-2\cdotK*M\cdot x+K*M^2-K*M^2)+N

=-\dfrac{1}{10*K}(x-K*M)^2+(K*M*M)/10+N

=-\dfrac{1}{10*K}(x-K*M)^2+Result

因此，當 x=K*M 時，y=Result 為最大值，也就是說，飛行過程中火箭離地面最高的高度為 Result 公尺 。

所以，本題答案為 \enspace (1)落地水平距離Res 公尺；\enspace(2)最大高度為Result公尺 。
