如上圖，矩形可以分為 S1, S2, S3, S4 四個部分，其面積為四個部分面積的和。

S1 = floor(L/10)*10\timesfloor(K/10)*10 = A

S2 = L%10\timesfloor(K/10)*10 = B

S3 = floor(L/10)*10\timesK%10 = C

S4 = L%10\timesK%10 = D

矩形面積 S = A+B+C+D = S
