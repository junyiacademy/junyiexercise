底面半徑為 r 、 高為 h 的圓柱體，它的表面積是多少？

init({ range: [[-1, 4], [-2, 4]] }); arc([1.5, 3], [1.5, 0.4], 0, 48); arc([1.5, 3], [1.5, 0.4], 70, 360); path([[0, -1], [0, 3]]); path([[3, -1], [3, 3]]); arc([1.5, -1], [1.5, 0.4], 0, 180, { strokeDasharray: "- " }); arc([1.5, -1], [1.5, 0.4], 180, 360); path([[1.5, 3], [3, 3]]); label([2.25, 3], r, "above"); label([3, 1], h, "right");

( 3.14 * 2 * r * ( r + h ) ).toFixed(2) 平方單位

上下兩個底面都是圓形，每個圓形面積是： = 半徑 \times 半徑 \times 3.14 \\= r \times r \times 3.14= r * r * 3.14。

側面的面積就跟高為 h 、寬和底面圓周一樣長的長方形面積一樣。

圓周長是 半徑 \times 2 \times 3.14 = r \times 2 \times 3.14 = 2 * r * 3.14 。

所以，它的側面面積是 h \times 2 * r * 3.14 = (2 * r * 3.14 * h).toFixed(2)

因此這個圓柱體的表面積是 r * r * 3.14 + r * r * 3.14 + ( 2 * r * h * 3.14 ).toFixed(2) = ( 3.14 * 2 * r * ( r + h ) ).toFixed(2)
