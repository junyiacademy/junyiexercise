橘 色線段是三角形的斜邊。另外兩邊 (如上圖的藍線)是abs( REAL ) 和 abs( IMAG )分別代表絕對值REPRESENTATION的實數與虛數部分。

代入畢氏定理a^2+b^2=c^2
\qquad |REPRESENTATION|^2 = abs( REAL )^2 + abs( IMAG )^2，所以
\qquad |REPRESENTATION| = \sqrt{abs( REAL )^2 + abs( IMAG )^2}。

\qquad \sqrt{abs(REAL)^2 + abs(IMAG)^2} = \sqrt{REAL * REAL + IMAG * IMAG} = \sqrt{ABS_SQUARE}

化簡這個根式，得到 formattedSquareRootOf( ABS_SQUARE )的絕對值是 REPRESENTATION。
