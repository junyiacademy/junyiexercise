坐標是表示為 (\blue{x},\green{y}).

所以，以 ( \blue{X}, \green{Y} )為例，我們的 x坐標 是 X 且 我們的 y坐標 是 Y.

x坐標 告訴我們從原點向右移多少 且 y坐標 告訴我們從原點向上移多少。

因為我們的 x坐標 等於 0，我不需要從原點向左移。

因為我們的 x坐標 等於 X, 我們從原點向右移 X 。

style({ stroke: BLUE, strokeWidth: 3, arrows: "->" }, function() { line( [ 0, 0 ], [ X, 0 ]); }); graph.movablePoint.toFront();
因為我們的 y坐標 等於 0，我不需要從原點向上移。

因為我們的 y坐標 等於 Y, 我們從原點向上移 Y。

style({ stroke: GREEN, strokeWidth: 3, arrows: "->" }, function() { line( [ X, 0 ], [ X, Y ] ); }); graph.movablePoint.toFront();
