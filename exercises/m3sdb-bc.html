邊長都相同，未必是正T邊形，舉例來說 ：

init({ range: [ [-8, 8], [-1, 9] ], scale: 30 }); path([[-7, 4],[-4, 0],[-1, 4],[-4, 8],[-7, 4]]); path([[1, 1.5],[6, 1.5],[6, 6.5],[1, 6.5],[1, 1.5]]);

以上兩個圖形， 每個邊長都是 randRange(1, 4) 公分，

左邊是菱形，右邊是正四邊形（正方形），

所以邊長相同，不一定是正多邊形。

角度都相同，未必是正T邊形，舉例來說 ：

init({ range: [ [0, 16], [-1, 9] ], scale: 30 }); path([[1, 1.5],[6, 1.5],[6, 6.5],[1, 6.5],[1, 1.5]]); line([1, 2], [1.5, 2]); line([1.5, 2], [1.5, 1.5]); line([5.5, 2], [6, 2]); line([5.5, 2], [5.5, 1.5]); line([1, 6], [1.5, 6]); line([1.5, 6.5], [1.5, 6]); line([5.5, 6.5], [5.5, 6]); line([5.5, 6], [6, 6]); path([[7, 1.5],[14, 1.5],[14 , 6.5],[7, 6.5],[7, 1.5]]); line([7, 2], [7.5, 2]); line([7.5, 2], [7.5, 1.5]); line([13.5, 2], [14, 2]); line([13.5, 2], [13.5, 1.5]); line([7, 6], [7.5, 6]); line([7.5, 6.5], [7.5, 6]); line([13.5, 6.5], [13.5, 6]); line([13.5, 6], [14, 6]);

以上兩個圖形， 每個角度都是 90 度，

左邊是正四邊形（正方形），右邊是長方形，

因此角度相同，不一定是正多邊形。

person(3)的說法是正確的。每個邊長一樣且角度都相同的多邊形，才是正T邊形。

正三角形（等邊三角形）是指一種三個邊均等長的三角形，其三個角大小相等，均為 60 度。

init({ range: [ [-5, 5], [-5, 4] ], scale: 30 }); path([[-3, -3], [3, -3], [0, 3 * sqrt(3) - 3], [-3, -3]]); arc([-3, -3], 1, 0, 60); arc([3, -3], 1, 120, 180); arc([0, 3 * sqrt(3) - 3], 1, 240, 300); label([-1.5, -2.3], '60' + '\^\\circ', { color : '#FFA500' }); label([1.5, -2.3], '60' + '\^\\circ', { color : '#FFA500' }); label([0, 3 * sqrt(3) - 4.5], '60' + '\^\\circ', { color : '#FFA500' }); label([-3, 0], '1公分', { color : '#6495ED' }); label([3, 0], '1公分', { color : '#6495ED' }); label([0, -4], '1公分', { color : '#6495ED' });

要小心與其他多邊形的觀念搞混，

只有在三角形時，若邊長相同或角度相同滿足任一條件，可確認正三角形。

如果是其他的多邊形，則是需要兩個條件同時滿足，才能是正多邊形。
