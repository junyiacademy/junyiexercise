randRangeUnique( 1, 10, 2 ).sort(function( a, b ){ return b - a; }) roundTo( 1, randRange( 0, 4 ) + _DECIMAL_A_LATER / 10 ) roundTo( 1, randRange( 0, floorTo( 0, _DECIMAL_A ) ) + _DECIMAL_B_LATER / 10 ) roundTo( 1, _DECIMAL_A - _DECIMAL_B )

_DECIMAL_A - _DECIMAL_B = ？

_ANSWER

做減法時，兩個數的位值要對齊。

所以寫成直式時，小數點剛好會上下對齊。要特別注意喔！

做減法時，要由右往左一個位值一個位值計算喔！

graph.adder = new DecimalSubtractor( roundTo( 0, _DECIMAL_A * 10 ), 1, roundTo( 0, _DECIMAL_B * 10 ), 1 ); graph.adder.show(); graph.adder.showDecimals();
graph.adder.showHint();

所以答案為 _ANSWER 。
