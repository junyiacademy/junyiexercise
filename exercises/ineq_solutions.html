單選+填充題

先解 - a x+b = c ， 不等式的最大/最小解就在等式的解的附近。 當x= x 時， - a x+b = c 等式成立。

取 x 附近的整數： x-1 、 x 、 x+1 來驗證： 當x= x-1 時， - a x + b = a*(x-1) + b = a*(x-1)+b = < > c (所以 x-1 不 是解)。 當x= x 時， - a x + b = a*(x) + b = a*(x)+b ≯ ≮ c (所以 x 不 是解)。 當x= x+1 時， - a x + b = a*(x+1) + b = a*(x+1)+b = < > c (所以 x+1 不 是解)。

因為 x-1 比 x 小， 所以 x-1 和比 x-1 小的整數都是不等式的整數解， x-1 是這些數中最大的數。

因為 x+1 比 x 大， 所以 x+1 和比 x+1 大的整數都是不等式的整數解， x+1 是這些數中最小的數。

所以答案是 最大整數解 = x-1 最小整數解 = x+1 。
