每當person(1)擁有NUM顆fruit( 0 )就擁有DENOM顆fruit( 1 ). 利用最簡分數寫出fruit( 0 )對fruit( 1 )的比例。

NUM / DENOM

比例可以用非常多不同形式來表達相同的意義。

你可以用冒號隔開兩個數字

NUM:DENOM

你可以用片語的方式來表達比例

NUM \text{ 比 } DENOM

或者，你可以用分數來表達比例

\dfrac{NUM}{DENOM}=fractionReduce( NUM, DENOM )

因此， fractionReduce( NUM, DENOM ) 是fruit( 0 )比fruit( 1 )的最簡分數表示式。
