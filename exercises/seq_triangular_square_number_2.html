randRange(5,30) N*(N+1)/2

試求 1 + 2+ 3 + \cdots + N 。

Ans

利用梯形公式 1+2+\cdots +n = \frac{n(n+1)}{2}。

將 n= N 代入，得到總和為 \frac{N\times N+1}{2} = Ans。

因此，本題的答案為 Ans 。

randRange(5,30) N*N

試求 1 + 3+ 5 + \cdots + 2*N-1 。

Ans

此數列公差為 2，第 n 項為 2n-1
所以 2*N-1 是此等差數列的第 N 項，所以項數 n=N。

利用梯形公式：
級數總和 =\frac{(1+2*N-1)\timesN}{2}

=\frac{2*N\times N}{2}=\frac{2\timesN\timesN}{2}=N^2。

因此，本題的答案為 N^2= Ans。
此外，我們可由此得到公式：
1 + 3+ 5 + \cdots + (2n-1) = n^2
